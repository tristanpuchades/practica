Navegadores web

Un navegador web (en inglés, web broser) es un software, aplicación o programa que permite el acceso a la Web, interpretndo la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador wen de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS

Mozilla Firefox es un navegador web libre y de código abierto desarrollado para Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporacion Mozilla y la fundacion Mozilla.

Google Chroome es un navegador web de codigo cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
